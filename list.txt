Створив папки admin, bin, config, додав різні файли. Створив кілька файлів з розширенням .sass , щоб у подальшому їх проігнорувати. 
Створив файл з розширенням .gitignore, а після цього приступив до вигрузки на віддалений репозиторій по наступними командами: 

git add .   Додаваня файлів до локального репозиторію

git status  Статус файлів, файли в очікуванні на вигрузку 

git commit -m "add files"   Вигрузка файлів до локального репозиторію.
